- 为了逐步带领大家了解ERD后端组成，会分三个层次逐步引导大家
- 第一步，点star后会加入到ERD单服务项目，该项目包含ERD运行时涉及到的全部接口

- 第三部，提交三个PR之后，会开放ERD整个后端基础架构权限
